# TenpennyMVC

A simple MVC framework built using PHP.

---

### Folder Structure

```bash
|-- app              # Contains all the frameworks Configuration, Controllers, etc.
    |-- libs         # Core files
    |-- models       # (M) Models
    |-- views        # (V) Views 
    |-- controllers  # (C) Controllers
    |-- helpers      # helper classes (modules)
    |-- config       # DB, Settings, etc
|-- public       # Contains all the public (static) files.
    |-- images          # All image files including static assets like icons etc.
    |-- javascripts     # All script files
    |-- stylesheets     # All CSS files
    |-- Vendor          # All Vendor libraries (jQuery, Bootstrap, etc)
```
